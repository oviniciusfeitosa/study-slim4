# Study Slim 4
Study case of slim framework 4


## Requirements 
- **composer require slim/slim:"4.*"*
- PHP 7.2+
- Composer
- php-mbstring

## Start

To run the application in development, you can run these commands 

```bash
composer start
```

or 

```bash
docker-compose up 
```
